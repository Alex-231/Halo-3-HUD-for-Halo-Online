IMPORTANT: Please backup your Halo folder, as this Pre-Release has been know to cause a few tag corruptions. 
Usually they're fixed pretty quickly but a backup always comes in handy.

To Install:
1. Download the zip.  https://github.com/Alex-231/Halo-3-HUD-for-Halo-Online
2. Download the Halo Online Tags Tool.   http://puu.sh/jG9W7/8cf83f74be.zip
3. Drag both the tag tool and the contents of the halo 3 HUD zip into your maps folder.
4. Run the installer batch file.

To uninstall just revert to your backup.
